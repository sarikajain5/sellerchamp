To run this project on Windows Mechine
Required Software
Operating System Windows-11
Java -java13
Selenium -4 
IDE -Eclipse
Build Tool Maven


Step to import the project

1. Keep SellerChamp prpoject in  'C' Drive
2. Import the project "SellerChamp"  by the File Menu of IDE Eclipse
3. Add TestNg plugin by the Eclipse market place to Eclipse IDE.
4. After succeessfully importing the project Did Maven Update by Right click on project and select Maven Update Command from Maven Menu.
5  Right click on Project, Maven Build
6. Right click on Project Select Run By TestNg
